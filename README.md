# Knee Osteoarthritis Severity Classification using Deep Learning

## Overview

This project is a deep learning based medical imaging system designed to detect and classify the severity of **Knee Osteoarthritis (OA)** from X-ray images.

The model analyzes knee X-ray scans and predicts the osteoarthritis severity level based on the **Kellgren-Lawrence grading scale (Grades 0–4)**. The objective is to assist in early diagnosis and provide a fast, accessible screening tool for patients and healthcare professionals.

This project was built as an end-to-end AI product pipeline including model training, evaluation, deployment, and a web interface for real-world testing.

---

## Problem Statement

Osteoarthritis is one of the most common joint disorders worldwide. Manual diagnosis through radiographic analysis requires trained specialists and can be time consuming.

The goal of this project is to automate the classification process by training a deep learning model capable of identifying the severity stage directly from X-ray images.

The model predicts five severity grades:

* Grade 0 — Healthy Joint
* Grade 1 — Doubtful Joint Space Narrowing
* Grade 2 — Mild Osteoarthritis
* Grade 3 — Moderate Osteoarthritis
* Grade 4 — Severe Osteoarthritis

---

## Dataset

The model was trained using a publicly available knee osteoarthritis X-ray dataset containing labeled knee radiographs categorized into five severity grades.

Dataset characteristics:

* X-ray based medical imaging dataset
* Five classification categories
* Class imbalance handled through augmentation and weighted sampling
* Images resized and normalized before training

Final class distribution after balancing:

* Class 0 → 2286 images
* Class 1 → 1400 images
* Class 2 → 1516 images
* Class 3 → 1400 images
* Class 4 → 1400 images

---

## Model Architecture

The final model uses **EfficientNet-B3** as the backbone architecture.

Architecture details:

* Pretrained EfficientNet-B3
* Transfer Learning from ImageNet weights
* Custom classification head
* Five output classes
* Fine tuning on full network

Training techniques used:

* WeightedRandomSampler for class imbalance
* Data augmentation
* Early stopping
* AdamW optimizer
* Learning rate scheduling
* CrossEntropyLoss with Label Smoothing

---

## Training Pipeline

Training configuration:

* Framework: PyTorch
* Batch Size: 32
* Optimizer: AdamW
* Learning Rate Scheduler: Cosine Annealing
* Epochs: 30
* GPU Acceleration using CUDA

Data augmentation:

* Horizontal Flip
* Rotation
* Affine Transformations
* Brightness and Contrast Jitter
* Image Normalization

## Model Performance

Final evaluation on unseen test dataset:

### Test Accuracy

68.84%

### Classification Report

| Grade | Precision | Recall | F1 Score |
| ----- | --------- | ------ | -------- |
| 0     | 0.72      | 0.89   | 0.80     |
| 1     | 0.35      | 0.21   | 0.26     |
| 2     | 0.69      | 0.64   | 0.67     |
| 3     | 0.79      | 0.81   | 0.80     |
| 4     | 0.79      | 0.86   | 0.82     |

Observations:

* Strong performance on severe osteoarthritis detection
* Moderate performance on middle severity grades
* Grade 1 remains the most difficult class due to visual similarity with neighboring grades
